# Пример локального MCP Gateway сервера для GigaAgent
В качестве тулов выбраны:
* Подключение к SQLite базе данных
* Отправка ботом сообщение в телеграм
* Пример получения изображения через MCP

Результаты работы GigaAgent с MCP-серверами, можно посмотреть здесь: [examples](examples)

> [!IMPORTANT]
> Для подключения из браузера к этому MCP-серверу мы отключаем в нем CORS заголовки. 
> Для большей безопасности подобных серверов, можно придумать систему аутентификации к ним (допустим через токен). 
> Сейчас такого здесь нет

## Установка
* `uv sync`
* В корне проекта заполните файл `.env` с переменными окружения (пример возьмите из [.env.example](.env.example))
* `uv run main.py`
* Теперь в GigaAgent вы можете подключить этот MCP через `http://127.0.0.1:6661/mcp`

## ENV-переменные
* **TELEGRAM_TOKEN** — токен для телеграм бота, который отправляет сообщения. Получить токена можно [тут](https://t.me/BotFather)
* **TELEGRAM_USER_ID** — user_id кому бот будет слать сообщения. Узнай id пользователя можно через этого [бота](https://t.me/userinfobot)
